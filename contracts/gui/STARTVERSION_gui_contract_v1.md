# STARTVERSION GUI Contract v1

## Purpose
Keep UI work consistent and low-risk.

## Rules
- Follow existing GUI guide/components; avoid ad-hoc visual systems.
- No new dashboard/board/html governance surfaces.
- UI changes must include clear behavior intent and validation steps.

## Acceptance minimum
- UX impact is stated.
- Affected user path is identified.
- Rollback for UI regression is documented.
